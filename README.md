# polishNotation

Convert strings of an infix equation to a string in prefix or [Polish Notation](https://en.wikipedia.org/wiki/Polish_notation).

Current output:
```
******************************
Infix Value : 3*(1+2)
Prefix Value: * 3 + 1 2
Your Answer : * 3 + 1 2
******************************
Infix Value : (3+5)*2
Prefix Value: * + 3 5 2
Your Answer : * + 3 5 2
******************************
Infix Value : (8+2)*(3+4)
Prefix Value: * + 8 2 + 3 4
Your Answer : * + 8 2 + 3 4
******************************
Infix Value : 4/(1+1)
Prefix Value: / 4 + 1 1
Your Answer : / 4 + 1 1
******************************
Infix Value : (3+1)/2
Prefix Value: / + 3 1 2
Your Answer : / + 3 1 2
```
